 2. Identify and break down LGWR wait events. Query wait events for LGWR.  
SELECT e.sid,
       e.event,
       e.time_waited,
       e.time_waited_micro
  FROM v$session_event e, v$session s
 WHERE e.sid = s.sid 
 AND s.program LIKE '%LGWR%'
order by 3;
